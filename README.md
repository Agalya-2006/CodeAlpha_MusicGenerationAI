# CodeAlpha_MusicGenerationAI
# Music Generation using AI

## Description
This project generates music using Artificial Intelligence. It processes MIDI files, extracts musical notes, and generates new music sequences.

## Technologies Used
- Python
- TensorFlow
- music21
- NumPy

## How it Works
1. Load MIDI files
2. Extract notes
3. Generate new music
4. Save as MIDI file

## How to Run
pip install -r requirements.txt
python train.py

## Output
The project generates an output.mid file which contains AI-generated music.

## Features
- Note extraction
- Music generation
- MIDI output creation
